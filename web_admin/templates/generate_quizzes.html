Informazioni sulla Generazione IA
Come funziona:

Il sistema utilizza l'API di OpenAI per generare automaticamente quiz educativi sul rugby. I quiz generati vengono poi salvati in una lista di attesa per essere revisionati e approvati dagli amministratori.

Processo di generazione:
  1. Seleziona il numero di quiz da generare
  2. Opzionalmente, scegli una categoria specifica
  3. Il sistema invia richieste all'API di OpenAI
  4. I quiz generati vengono salvati nella lista di attesa
  5. Gli amministratori possono approvarli o rifiutarli
Nota: La generazione di quiz richiede una chiave API di OpenAI valida configurata nel file .env.
Suggerimenti:
  • Genera pochi quiz alla volta per una revisione più efficiente
  • Controlla sempre l'accuratezza delle informazioni generate
  • Modifica i quiz se necessario prima di approvarli
